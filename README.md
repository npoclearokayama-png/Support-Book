# サポートブック作成ツール

発達支援・就学支援で共有しやすい「サポートブック」を、ブラウザ上で編集・保存・PDF出力できるシングルページアプリです。

## できること
- 5つのタブ（表紙 / 長所・興味 / 伝え方 / ここで困るかも / 場面別）で情報を整理
- 写真の追加・移動・拡大縮小・回転
- 端末内保存（LocalStorage）
- JSON エクスポート / インポート
- A4レイアウトで印刷またはPDF出力

## ファイル構成
- `index.html` : 画面の骨組み（DOM）
- `assets/styles.css` : 画面スタイル
- `assets/app.js` : 状態管理、編集UI、ページネーション、出力処理

## ローカルでの使い方
1. このフォルダを開く
2. `index.html` をブラウザで開く
3. 入力した内容は自動保存されます

> モバイル端末では、機種やブラウザによってPDF保存の挙動が異なる場合があります。
